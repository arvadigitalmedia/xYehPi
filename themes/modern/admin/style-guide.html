Usage Guidelines

Accessibility
  • Minimum touch target size: 44px × 44px
  • Text contrast ratio: ≥7:1 for normal text, ≥4.5:1 for large text
  • Focus ring: 2px solid gold-400 with 2px offset
  • All interactive elements must have proper ARIA labels
  • Keyboard navigation support for all components
Responsive Breakpoints
  • Desktop: 1440px (primary design target)
  • Laptop: 1280px
  • Tablet: 1024px
  • Mobile: 768px
  • Small Mobile: 390px
Animation & Transitions
  • Fast: 150ms ease-in-out (hover states)
  • Normal: 200ms ease-in-out (general transitions)
  • Slow: 300ms ease-in-out (complex animations)
  • Active state: scale(0.99) for button press feedback
  • Hover elevation: translateY(-2px) for cards
