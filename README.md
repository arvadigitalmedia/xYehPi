## ğŸš€ Installation

### Quick Start

1. **Clone Repository**
   ```bash
   git clone https://github.com/arva/epic-hub.git
   cd epic-hub
   ```

2. **Install Dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Environment Setup**
   ```bash
   cp .env.example .env
   # Edit .env file dengan konfigurasi database dan aplikasi
   ```

4. **Database Setup**
   ```bash
   php artisan migrate
   php artisan db:seed
   ```

5. **Build Assets**
   ```bash
   npm run build
   ```

6. **Start Development Server**
   ```bash
   php artisan serve
   ```

### Docker Installation

```bash
docker-compose up -d
```
